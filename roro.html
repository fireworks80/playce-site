Playce RoRo는
Linux / Unix 로 구축된
AS-IS 인프라를 클라우드로
전환해주는 솔루션입니다.

자동화된 시스템 진단 분석 기능을 바탕으로 AS-IS 인프라를 클라우드로 보다 빠르고 안전하게 이동할 수 있습니다.

특장점

Why Playce RoRo?

다양한 OS 지원
  • Red Hat 및 Debian 계열의 Linux에 대한
    클라우드 마이그레이션 지원
  • AIX Unix의 클라우드 마이그레이션 지원
비용절감
  • 웹 콘솔 기반 마이그레이션 마법사를 통한
    소요 비용(인력/시간) 절감
  • 무중단 마이그레이션으로 서비스의 다운
    타임이 최소화됨
유연하고 안정적인 시스템
  • 마이그레이션 대상 서버에 대한 유연한
    구성 가능
  • 마이그레이션 대상 서버에 구성한 설정과
    소프트웨어의 변경 없이 보존 가능

가이드 영상

쉽고 빠른 클라우드 마이그레이션을 경험해 보세요

주요기능

L2C 및 U2C 마이그레이션을 효과적으로 수행하면서 유연한 볼륨 구성, 네트워크 설정 지원, 자동 이미지 생성 등 다양한 기능을 제공합니다

마이그레이션 Wizard 기능

  1. Step 0. start
  2. Step 1. 마이그레이션 수행 라이선스 선택
  3. Step 2. 마이그레이션 대상 소스 서버 선택
  4. Step 3. 타깃 클라우드(AWS)에 대한 네트워크 설정
  5. Step 4. 타깃 인스턴스의 타입 선택
  6. Step 5. 타깃 인스턴스의 볼륨 정보 설정
  7. Step 6. Wizard step 검토 및 마이그레이션 실행
Agentless 마이그레이션 도구
  • Agent의 설치 없는 웹 콘솔 기반의 SaaS 서비스
  • 서버에 대한 최소한의 정보 등록으로 마이그레이션 환경 구성 가능
Path별 유연한 볼륨 구성
  • 온프레미스의 단일 볼륨을 여러 개의 EBS 볼륨으로
    분리 가능한 기능 제공
  • 온프레미스의 여러 볼륨을 단일 EBS 볼륨으로 통합
    가능한 기능 제공
Network 설정 지원
  • AWS Console 접속 없이 타깃 클라우드에 대한 Access
    Key 및 Region 설정 기능
  • 네트워크 관련 VPC, Subnet, 그리고 Security Group에
    대한 설정 지원
APP 및 워크로드 마이그레이션
  • 기존 온프레미스에 구성한 설정과 소프트웨어의
    변경 없이 마이그레이션 가능
자동 이미지 생성
  • 백업 및 재해 복구용으로 마이그레이션 시점의
    온프레미스 이미지를 자동으로 생성
미들웨어(WAS) 분석 기능
  • WebLogic, WebSphere, Jeus에서 작동하는
    웹 애플리케이션의 마이그레이션 리포트 생성 및 변환

아키텍처

자동화된 시스템 분석 도구 및 마이그레이션 모듈을 이용하여
온프레미스 서버를 이미지화, 클라우드 저장소에 저장하고
이미지로부터 인스턴스와 추가되는 볼륨을 생성합니다

표준 지원 명세

지원환경은 아래 표의 정보를 참고해주세요

System

Network
  • RoRo에서 사용자의 서버로 SSH 접속이 가능해야 함
  • 필요한 경우 VPN을 사용할 수 있으며, 별도의 비용이 발생가능
SSH 접속 가능한 사용자 계정
  • 패스워드를 이용한 로그인이 가능해야 함
  • 패스워드 확인 없이 sudo 명령이 실행 가능해야 함
패키지
  • rsync (리눅스에만 필요)
  • python 2.7+ (리눅스에만 필요)
  • Java 1.7+ (WAS 분석 기능 사용 시에만 필요)

Migration

Migration
  • 마이그레이션을 수행하기 위해 사전에 라이선스 구매가 필요
AWS Service Limit
  • Region 당 동시에 최대 5개의 마이그레이션만 수행 가능
  • 필요한 경우 Support Center를 통해 Limit 상향 조정해야 함
