Playce WASup

Playce WASup을 무료로 사용할 수 있는 Trial Key를 제공합니다. Trial Key는 기능상의 제한이 없으며 최신 버전으로 사용 가능합니다.

  1. Step 1. Trial Key 신청

    • Playce WASup을 30일간 무상으로 사용하실 수 있는 Trial Key를 제공합니다.

    • Trial Key는 별도의 신청 없이 설치파일을 통해 제공됩니다.

  2. Step 2. 설치파일 다운로드

    • 설치파일은 tar.gz, zip 파일 형식으로 제공됩니다.

      tar.gz Zip

    • 설치파일을 다운로드 받으시면 소프트웨어 사용권 에 동의하신 것으로 간주합니다.

    * PC 버전만 다운로드 가능합니다.

  3. Step 3. 접속 및 사용

    • Playce WASup을 설치하고 사용하기 위해 설치 가이드를 참조해 주세요.

      설치 가이드 Installation guide

    • Trial Key 적용 방법은 Subscription 갱신 가이드를 참조해 주세요.

      서브스크립션 갱신 가이드

Playce RoRo

Playce RoRo를 무료로 사용할 수 있는 Trial license를 제공합니다. Trial license는 기능상의 제한이 없으며 설정된 만료일까지 무료로 사용할 수 있습니다.

  1. Step 1. 회원가입하기

    • Playce RoRo 콘솔로 이동하여 회원가입을 해야 합니다.

      Playce RoRo 콘솔

    • Facebook, Google 계정을 이용한 회원가입도 가능합니다.

  2. Step 2.Trial license 신청

    • ‘Trial license 신청’ 버튼을 클릭하시어 Trial license를 신청해 주세요.

      Trial license 신청 Trial license request

    • Trial license는 담당자의 확인 후 24시간 안에 메일로 보내드립니다.

  3. Step 3. 접속 및 사용

    • Playce RoRo 접속하시어 Trial license를 적용 후 바로 사용이 가능합니다.

      사용자 가이드 User guide

    • Trial license 적용 방법은 사용자 가이드를 참조해 주세요.
